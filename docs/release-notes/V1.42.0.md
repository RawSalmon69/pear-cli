### Changelog

1. **pe clean**: Project artifacts scan now finishes on huge workspaces and shows partial results when slow scans time out
2. **pe clean**: Typing a sudo password no longer gets treated as skipping sudo
3. **pe clean**: App leftovers can remove dead macOS app records from deleted apps and unmounted DMGs without deleting files
4. **pe clean**: Developer cleanup finds more rebuildable caches, including VS Code WebStorage, QQ, LM Studio, Folo, SenPlayer, Xcode simulator caches, and XCTest test data
5. **pe analyze**: Cached folders keep their results visible while Pear refreshes in the background
6. **pe update**: Nightly updates skip reinstalling the same build
7. **pe uninstall**: Related-file cleanup is stricter around system items, app helpers, symlinks, and container caches

### 更新日志

1. **pe clean**：Project artifacts 在超大工作区里不会再卡住，慢扫描超时后仍会展示已找到的结果
2. **pe clean**：在 sudo 提示里输入密码不会再被误判成跳过 sudo
3. **pe clean**：App leftovers 可以清理已删除应用、已卸载 DMG 留下的失效应用记录，且不会删除文件
4. **pe clean**：Developer cleanup 能发现更多可再生成缓存，包括 VS Code WebStorage、QQ、LM Studio、Folo、SenPlayer、Xcode 模拟器缓存和 XCTest 测试数据
5. **pe analyze**：打开已有缓存的文件夹时，后台刷新期间会继续显示已有结果
6. **pe update**：Nightly 更新不再重复安装同一个版本
7. **pe uninstall**：相关文件清理在系统项、应用 helper、symlink 和 container cache 上更谨慎
