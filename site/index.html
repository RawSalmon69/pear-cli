What’s inside

Small tools, done the native way.

Each one leans on Apple’s own frameworks instead of reinventing them — so it stays fast, quiet and trustworthy.

Private by architecture

Pear has no server to
send your data to.

On-device, always

Text recognition, background removal and QR scanning run on your Mac through Apple’s Vision framework. Nothing leaves the machine.

No accounts, no telemetry

Nothing to sign into, nothing phoning home. There isn’t an analytics pipeline because there isn’t a pipeline.

Signed & notarized

Every release is built in CI, signed with a Developer ID and notarized by Apple before it ships.

Quiet updates

Pear updates itself over Sparkle from a signed feed — the mechanism trusted Mac apps have used for years.

One optional download

High-quality background removal fetches a ~233 MB model only if you turn it on. It’s removable, and the default works without it.

Pear is named after someone dear. It started as a gift, grew into a toolbox, and is shared the way homemade things are — free, for friends, family, and now you.

No pricing page. No roadmap. Just a small thing, made carefully.
