RMBG-2.0 background-removal model (High-quality mode)
=====================================================

Pear Companion's optional "High-quality mode" for background removal downloads
and runs, on-device, a Core ML conversion of BRIA AI's RMBG-2.0 model. The model
is NOT bundled with the app; it is fetched on demand only if the user opts in,
and can be removed at any time from Settings.

Core ML conversion:
  VincentGOURBIN/RMBG-2-CoreML — https://huggingface.co/VincentGOURBIN/RMBG-2-CoreML
  License: CC-BY-NC-4.0 (Creative Commons Attribution-NonCommercial 4.0)

Base model:
  BRIA AI RMBG-2.0 — https://huggingface.co/briaai/RMBG-2.0
  Architecture: BiRefNet. Released for NON-COMMERCIAL use; commercial use
  requires a separate license agreement with BRIA AI.

NON-COMMERCIAL USE ONLY. This feature must not be shipped in a commercial
product without obtaining the appropriate commercial license from BRIA AI.
Pear's built-in background removal (Apple Vision) has no such restriction and
remains the default.
